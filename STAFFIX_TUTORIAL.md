# Staffix — Краткое руководство по настройке

## Что такое Staffix?

Staffix — AI-платформа, которая создаёт умного Telegram-бота для вашего бизнеса.
Бот автоматически записывает клиентов, принимает заказы, отвечает на вопросы и уведомляет вас о новых обращениях.

---

## Шаг 1 — Регистрация

1. Откройте [staffix.io](https://staffix.io)
2. Нажмите «Начать бесплатно»
3. Войдите через Google или Email
4. Выберите тип бизнеса: **Услуги** (салон, клиника, мастер) или **Магазин** (товары, заказы)

> Trial период: **14 дней бесплатно**, 100 сообщений

---

## Шаг 2 — Создание Telegram-бота

1. Откройте Telegram → найдите **@BotFather**
2. Отправьте `/newbot`
3. Придумайте имя бота (например: «Салон Красоты Анна»)
4. Придумайте username (например: `salon_anna_bot`)
5. Скопируйте **токен** (выглядит так: `7123456789:AAFxxx...`)

В Staffix → **Мой бот** → вставьте токен → нажмите «Подключить»

---

## Шаг 3 — Добавление сотрудников (для услуг)

1. Dashboard → **Команда** → «Новый сотрудник»
2. Заполните: имя, должность, график работы
3. Укажите Telegram-username сотрудника (он получит уведомления о записях)
4. Если сотрудников много — загрузите через **Импорт CSV**:

```
Имя;Должность;График
Анна Иванова;Мастер маникюра;Пн-Пт 9:00-18:00
```

---

## Шаг 4 — Добавление услуг или товаров

### Услуги (через CSV)
Dashboard → **Мои услуги** → «Импорт CSV»

Формат файла:
```
Название;Цена;Длительность (мин);Описание
Маникюр классический;3000;60;Базовое покрытие
Маникюр с гель-лаком;4500;90;Стойкое покрытие
```

### Товары (через CSV)
Dashboard → **Мои товары** → «Импорт CSV»

Формат файла:
```
Название;Цена;Категория;Описание;Остаток;Артикул;Старая цена
iPhone 15;89990;Смартфоны;128 GB Black;10;IP15-BLK;99990
```

> Только первые два поля обязательны (название + цена)

---

## Шаг 5 — База знаний (что знает ваш бот)

1. Dashboard → **База знаний** → «Добавить»
2. Загрузите документы: FAQ, прайс-лист, правила, адрес
3. Поддерживаемые форматы: PDF, DOCX, TXT (до 10 МБ)

**Или напишите вручную:**
- Адрес и режим работы
- Политика отмены записей
- Условия доставки/возврата
- Часто задаваемые вопросы

> Чем больше информации — тем умнее бот

---

## Шаг 6 — Настройка AI-сотрудника

Dashboard → **Мой бот** → раздел «AI-сотрудник»

1. Выберите **промпт-шаблон** под ваш тип бизнеса
2. Или напишите свой промпт, например:

```
Ты — администратор салона красоты «Бьюти». Записывай клиентов
на услуги, отвечай на вопросы о ценах и времени работы.
Всегда будь вежлив и профессионален. Предлагай доступные слоты.
```

3. Укажите ваш Telegram для уведомлений (бот будет присылать вам каждое новое обращение)

---

## Шаг 7 — Тест и запуск

1. Откройте вашего бота в Telegram
2. Напишите «Привет» — бот должен ответить
3. Попробуйте записаться / заказать товар
4. Проверьте что уведомление пришло вам в Telegram

### Что должно работать:
- ✅ Бот отвечает на вопросы
- ✅ Запись/заказ создаётся в Dashboard
- ✅ Вы получаете уведомление в Telegram
- ✅ Клиент получает подтверждение

---

## Уведомления

| Событие | Кто получает |
|---------|-------------|
| Новая запись | Вы + сотрудник |
| Новый заказ | Вы |
| Изменение статуса заказа | Клиент |
| Отмена записи | Вы + сотрудник |
| Бот не может ответить | Вы (эскалация) |

---

## Настройка оплаты (опционально)

Dashboard → **Мой бот** → «Оплата»

Поддерживаемые платёжные системы:
- **Payme** — введите Merchant ID
- **Click** — введите Service ID + Merchant ID
- **Kaspi Pay** — введите токен

После подключения бот будет автоматически предлагать оплату в Telegram.

---

## Статусы заказов

При изменении статуса в Dashboard → клиент автоматически получает сообщение:

| Статус | Сообщение клиенту |
|--------|------------------|
| Подтверждён | ✅ Ваш заказ подтверждён! |
| В обработке | ⚙️ Ваш заказ в обработке |
| Отправлен | 🚚 Ваш заказ отправлен! |
| Доставлен | 🎉 Ваш заказ доставлен! |
| Отменён | ❌ Ваш заказ отменён |

## Частые вопросы

**Сколько времени занимает настройка?**
При готовых данных (список услуг/товаров, FAQ) — **15-30 минут**.

**Нужны ли технические знания?**
Нет. Всё делается через интерфейс, без кода.

**Что если бот не знает ответа?**
Бот уведомит вас и предложит клиенту связаться напрямую. Разговор продолжится.

**Можно ли подключить WhatsApp?**
WhatsApp и Facebook Messenger — в разработке, скоро будет доступно.

**Как добавить больше сотрудников или услуг позже?**
В любой момент через Dashboard. Бот автоматически получит новые данные.

# Казахстан — пакеты для внедрения Staffix (партнёр по платёжным системам)

Материалы по 26 клиентам партнёра из РК для запуска комплексного продукта «эквайринг + AI-сотрудник Staffix». Подготовлено 1 июля 2026.

## Что здесь

- **`_АНАЛИЗ/00_Сводный_анализ_клиентов.md`** — анализ каждого клиента, сводная таблица, группировка по сценариям, флаги/риски, порядок захода. **Начать чтение отсюда.**
- **`_АНАЛИЗ/01_Модель_себестоимости.md`** — расчёт себестоимости Staffix на клиента (1000 сообщений/мес), диапазоны, юнит-экономика.
- **`NN_Название/`** — по папке на клиента. В каждой:
  - `Услуги.csv` — готов к импорту в Staffix (Дашборд → Мои услуги → Импорт CSV). Формат `Название;Цена;Длительность;Описание`, тенге.
  - `FAQ.md` — вопросы/ответы для базы знаний (Дашборд → База знаний / FAQ).
  - `Правила_и_настройки.md` — заготовка настроек AI-сотрудника (имя, тон, приветствие, часы, правила, каналы).
  - `README_источники.md` — источники данных, что реально/ориентир, флаги.

## Как загружать в Staffix (по коду платформы)

1. **Услуги** — `/dashboard/services` → «Импорт CSV». Распознаёт `;` и `,`, до 10 000 строк.
2. **FAQ / база знаний** — `/dashboard/faq` (вопрос/ответ) или загрузка документа в Базу знаний.
3. **Правила и тон** — `/dashboard/settings` (aiRules, aiTone, приветствие, часы) и `/dashboard/bot` (имя бота).
4. **Каналы** — `/dashboard/channels`: Telegram (токен), Instagram/WhatsApp/Facebook (через Meta OAuth). Приоритет подключения: Instagram → WhatsApp → Telegram.

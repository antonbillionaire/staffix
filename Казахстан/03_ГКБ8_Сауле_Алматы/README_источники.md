# README — источники и статус данных

## Учреждение
- **Название:** Городская клиническая больница №8 (ГКБ №8), бывшая железнодорожная больница
- **Тип:** многопрофильный стационар и поликлиника (терапия, хирургия, урология, гинекология, неврология, реабилитация; диагностика — МРТ, УЗИ, ЭКГ, рентген, лаборатория)
- **Орг.форма:** НЕ ясна — предположительно государственное ГКП, но возможен формат ТОО. Требует уточнения.
- **Город:** Алматы
- **Адрес:** ул. Акан Серы, 36А

## Источники
- Сайт: almaty-gkb8.kz (основной источник)
- Instagram: @almaty_rw.kz
- Онлайн-запись: egov.kz

## Статус цен
⚠️ **ВСЕ цены в файле Услуги.csv — ОРИЕНТИР.** Прайс-лист не найден: на сайте actуального прайса нет. Стоимости взяты из отраслевых ориентиров по Алматы 2026. В каждом описании услуги стоит пометка «(ориентир — заменить на прайс клиники)». Перед запуском бота цены необходимо заменить на реальный прайс клиники.

## Что уточнить у клиента / партнёра
1. Актуальный прайс-лист на платные услуги (все цены сейчас ориентировочные).
2. Точные часы работы поликлиники, диагностики и отдельных специалистов.
3. Организационно-правовую форму (ГКП или ТОО) и точное юридическое название.
4. Перечень услуг, реально оказываемых больницей (список в CSV — предположительный).
5. Какие услуги доступны бесплатно по ОСМС/прикреплению, а какие только платно.
6. Контактные каналы: телефон регистратуры, WhatsApp, Telegram (сейчас не указаны).
7. Наличие и параметры МРТ (тип аппарата, доступные зоны исследования).

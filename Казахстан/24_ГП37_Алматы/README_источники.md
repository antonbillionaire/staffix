# README — источники и статус данных (24_ГП37_Алматы)

## Субъект
Городская поликлиника №37, г. Алматы, Бостандыкский район.
- ГОСУДАРСТВЕННОЕ учреждение ПМСП (многопрофильная поликлиника).
- Новое здание на ~500 посещений/смену.

## Источники
- Данные предоставлены партнёром.
- Отдельного сайта, соцсетей и прайса у поликлиники НЕТ.
- Профиль услуг взят ПО АНАЛОГИИ с ГП №17 (многопрофильная поликлиника ПМСП).

## Реальные данные vs ориентир
- РЕАЛЬНЫЕ (частично): название, район, статус государственной поликлиники.

## Специальные флаги
- **Город Астана:** все настройки времени и адреса — астанинские. Не путать с другими городами.
- **Call-центр 64-07-11 — узкое горло записи (главная боль клиента):** единый номер загружен, пациенты не всегда дозваниваются. Ключевой сценарий бота — разгрузка call-центра: информирование по услугам/расписанию/подготовке и предварительный сбор данных для записи (ФИО, телефон, услуга/специалист, удобное время).
- **Прайс не подтверждён:** все цены помечены как ориентир, бот обязан предупреждать об этом.
- **Не выдумывать:** врачи поимённо и точные цены не указываются, пока не подтверждены клиентом.

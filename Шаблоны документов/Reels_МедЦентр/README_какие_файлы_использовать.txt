============================================================
ФАЙЛЫ ДЛЯ ЗАПИСИ РОЛИКОВ — МЕДЦЕНТР
============================================================

В этой папке 4 файла для съёмки 22 роликов.

============================================================
СПИСОК ФАЙЛОВ
============================================================

1. 00_БАЗА_ЗНАНИЙ_МЕДЦЕНТРА.txt
   ГДЕ ИСПОЛЬЗОВАТЬ: загрузить ОДИН РАЗ перед всеми роликами
   КУДА: Дашборд → База знаний → "Загрузить файл"
   ЗАЧЕМ: AI узнает весь профиль медцентра — услуги, врачей,
   пакеты, FAQ, правила. После этого AI может отвечать корректно
   во всех роликах.

2. 01_Прайс_Медцентра_Стаффикс_Мед.pdf
   ГДЕ ИСПОЛЬЗОВАТЬ: ролик 2.2 "База знаний — загрузите и AI выучит"
   КУДА: Дашборд → База знаний → "Загрузить файл"
   ЗАЧЕМ: на видео показываете загрузку реального PDF-прайса.
   Демонстрирует что Staffix распознаёт цены из PDF.

3. 02_Услуги_для_импорта.xlsx
   ГДЕ ИСПОЛЬЗОВАТЬ: ролик 2.3 "Услуги и пакеты со скидкой"
   КУДА: Дашборд → Мои Услуги → "Импорт CSV"
   ЗАЧЕМ: на видео показываете массовый импорт всех 47 услуг
   медцентра из Excel за один клик. Экономит время съёмки —
   не нужно добавлять услуги вручную.

4. 02b_Услуги_для_импорта.csv (резервный)
   Используйте если Excel не загрузится (например на другом
   устройстве). Содержит 10 ключевых услуг для быстрой демо.

============================================================
ПОРЯДОК ПОДГОТОВКИ К СЪЁМКАМ
============================================================

ШАГ 1: Зарегистрировать (или открыть) демо-аккаунт "Медцентр Стаффикс"
ШАГ 2: Загрузить 00_БАЗА_ЗНАНИЙ_МЕДЦЕНТРА.txt в Базу знаний
ШАГ 3: Импортировать услуги через 02_Услуги_для_импорта.xlsx
ШАГ 4: Создать 6 пакетов услуг вручную (по сценариям из Word):
       - Check-up Базовый
       - Check-up Женский
       - Check-up Мужской
       - Check-up Сердце
       - Пакет "Брови Идеал"
       - Пакет "Весеннее обновление"
ШАГ 5: Создать 8 правил несовместимости (по БЗ)
ШАГ 6: Добавить врачей с расписанием (10 человек)
ШАГ 7: Создать 5-7 тестовых пациентов с разной активностью
ШАГ 8: Включить автоматизации (напоминания, отзывы, реактивация)

После этого можно записывать любой из 22 роликов.

============================================================
ВАЖНО
============================================================

Файл 01_Прайс PDF — это та же информация что в 00_БАЗА_ЗНАНИЙ.txt,
но в красивом PDF-формате. Для ролика 2.2 нужно показать загрузку
именно PDF (визуально убедительнее чем txt).

После загрузки PDF в Базу знаний — Staffix распознает текст и
добавит к существующим знаниям. Дублирования не произойдёт —
AI просто получит дополнительный источник информации.

Файл 02_Услуги.xlsx содержит ВСЕ 47 услуг медцентра. После импорта
у вас сразу будет полный каталог услуг — можно сразу записывать
ролики 2.3, 2.4, 2.5, 3.1 и далее.
